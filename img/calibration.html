  1. Using Distribution of Employment from SIC Divisions to NAICS Supersectors (in thousands), link SIC divisions in BDS data to NAICS supersectors. Notice that this shares data was created using 2001 data.

  2. Using Current Employment Statistics (Establishment Survey) data, link NAICS supersectors to NAICS 2-Digit industries.

  3. Using BEA I-O and NAICS crosswalk, link 2-Digit NAICS industries to I-O data.

Following function calculates consumption share of each industries output while taking intermediate inputs into account. Methodology is from Sector-Specific Technological Change by Basu, Feranld, Fisher, and Kimball (2013).

Consumption share of each industry is $\phi = (I-B)^{-1}\psi$, where B is the I-O matrix, and $B_{i,j}$ is share of commodity i output used in industry j production. $\psi$ is the share of commodity $i$ output consumed by households.

def cons_inv_int_share(group):
    # create matrix
    single_year = group.pivot(index='RowCode',columns='ColCode',values='DataValue').fillna(0)
    # convert I-O matrix into shares with each row sums to 1 across columns.
    single_year_shares = single_year.div(single_year.sum(axis=1),axis=0)
    # make sure that row and columns industries are ordered the same
    m=list(single_year_shares.index.values)
    n= m+list(np.setdiff1d(single_year_shares.columns,m))
    single_year_share = single_year_shares[n]
    # formula of this described above
    cons_share = (np.linalg.inv(np.identity(len(single_year_share))
                                -single_year_share[m].values).dot(single_year_share['F010'].values))
    # investment share is 1 - consumption share
    inv_share = 1-cons_share
    # create a dataframe to return
    result = pd.DataFrame(index=m,data={'Consumption Share':cons_share,'Investment Share':inv_share})
    result.index.name = 'IOCode'
    return result

After I compiled the data set, I can now calculate moments that will be targetted in my calibration. These are job creation rate by entrants and job creation rate by incumbents in consumption goods producing and investment goods producing sectors. Job creation by consumption goods is calculated by

  1. for each industry-year pair, multiplying job creation by the share of output of that industry going into consumption
  2. summing over industries the above figure.

The same exercise is repeated for both consumption and investment sectors and different data types: employment and job creation by incumbents.

def calc_cons_invs_sector(group):
    # calculate employment in consumption and investment sectors by multiplying employment 
    # with consumption and investment shares and summing over industries
    emp_cons = (group.emp*group['Consumption Share']).sum()
    emp_inv = (group.emp*group['Investment Share']).sum()
    # similarly calculate job creation by birth, job creation by expansion
    # for both consumption and investment industry
    job_creation_birth_cons = (group.job_creation_births*group['Consumption Share']).sum()
    job_creation_birth_inv = (group.job_creation_births*group['Investment Share']).sum()
    job_creation_continuers_cons = (group.job_creation_continuers*group['Consumption Share']).sum()
    job_creation_continuers_inv = (group.job_creation_continuers*group['Investment Share']).sum()
    job_creation_rate_birth_cons = job_creation_birth_cons/emp_cons
    job_creation_rate_birth_inv = job_creation_birth_inv/emp_inv
    job_creation_rate_continuers_cons = job_creation_continuers_cons/emp_cons
    job_creation_rate_continuers_inv = job_creation_continuers_inv/emp_inv
    
    job_destruction_death_cons = (group.job_destruction_deaths*group['Consumption Share']).sum()
    job_destruction_death_inv = (group.job_destruction_deaths*group['Investment Share']).sum()
    job_destruction_continuers_cons = (group.job_destruction_continuer*group['Consumption Share']).sum()
    job_destruction_continuers_inv = (group.job_destruction_continuer*group['Investment Share']).sum()
    job_destruction_rate_death_cons = job_destruction_death_cons/emp_cons
    job_destruction_rate_death_inv = job_destruction_death_inv/emp_inv
    job_destruction_rate_continuers_cons = job_destruction_continuers_cons/emp_cons
    job_destruction_rate_continuers_inv = job_destruction_continuers_inv/emp_inv

    return pd.Series({'emp_cons':emp_cons,
                     'emp_inv':emp_inv,
                     'job_creation_rate_birth_cons':job_creation_rate_birth_cons,
                     'job_creation_rate_birth_inv':job_creation_rate_birth_inv,
                     'job_creation_rate_continuers_cons':job_creation_rate_continuers_cons,
                     'job_creation_rate_continuers_inv':job_creation_rate_continuers_inv,
                     'job_destruction_rate_death_cons':job_destruction_rate_death_cons,
                     'job_destruction_rate_death_inv':job_destruction_rate_death_inv,
                     'job_destruction_rate_continuers_cons':job_destruction_rate_continuers_cons,
                     'job_destruction_rate_continuers_inv':job_destruction_rate_continuers_inv})

def labor_share(group):
    group = group.dropna(subset=['labor compensation','Consumption Share','Investment Share',
                                 'Taxes on production and imports less subsidies','Value added'])
    labor_comp_con = (group['labor compensation']*group['Consumption Share']).sum()
    labor_comp_inv = (group['labor compensation']*group['Investment Share']).sum()
    taxes_con = (group['Taxes on production and imports less subsidies']*group['Consumption Share']).sum()
    taxes_inv = (group['Taxes on production and imports less subsidies']*group['Investment Share']).sum()
    value_added_con = (group['Value added']*group['Consumption Share']).sum()
    value_added_inv = (group['Value added']*group['Investment Share']).sum()
    
    labor_share_con = labor_comp_con/(value_added_con-taxes_con)
    labor_share_inv = labor_comp_inv/(value_added_inv-taxes_inv)

    return pd.Series({'labor_share_cons':labor_share_con,
                     'labor_share_inv':labor_share_inv})

def labor_prod_growth(group):
    select_prod = ((group['Measure'] == 'Labor productivity') & (group['Units'] == '1-Yr Rate'))
    select_emp = ((group['Measure'] == 'Number of employees') & (group['Units'] == 'thousands'))
    prod = np.array(group[select_prod]['Value'].values)
    emp = np.array(group[select_emp]['Value'].values)
    emp_sum = np.sum(emp)
    return pd.Series({'prod_growth':np.sum(prod*emp/emp_sum),'total_emp':emp_sum})
